How to fix to FHD

1. Download Return To Castle Wolfenstein WideScreen Fix 

2. Extract anywhere

3. Copy/Paste into your Return To Castle Wolfenstein Steam or GoG directory

4. Set the resolution to 1920x1080 in game settings 

5. Enjoy!
